OOP Inventory Management System
A streamlined C++ application for managing inventory using Object-Oriented Programming principles.

 Features
CRUD Operations: Add, remove, and list inventory items.

Stock Management: Track quantities, IDs, and pricing.

UML Design: Includes a class diagram for architectural clarity.

Technical Report: Full project documentation included as a PDF.

 Quick Start
Compile and run the project using any C++ compiler:

Bash

g++ main.cpp Item.cpp InventoryManager.cpp -o InventorySystem
./InventorySystem
Core Files
Item.h/cpp: Product data model.

InventoryManager.h/cpp: Core logic and storage.

main.cpp: CLI menu and user interaction.

UML_CLASS_DIAGRAMM.png: Visual architecture.
